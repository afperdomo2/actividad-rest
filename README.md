## 💻Project Setup

```sh
npm install
```

### Run Development

```sh
npm run dev
```

### Run Production

```sh
# Ejecutar el entorno de desarrollo
npm run dev

# Ejecutar el entorno de producción
npm run start
```

### Run linter

```sh
npm run lint
```

## 🐳 Docker Compose

```sh
# Crear el contenedor de postgres
docker-compose up -d postgres
```
